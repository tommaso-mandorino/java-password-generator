# Esercizio "Password Generator"

## Passaggi
1. Creare un nuovo progetto Java chiamato `java-password-generator`.
2. Aggiungere un package `org.lessons.java.security`.
3. Aggiungete una classe `PasswordGenerator` che contenga un metodo `main`.

## Richieste
1. Salvare in opportune variabili il **nome**, **cognome**, **colore preferito** e **data di nascita** di un utente suddivisa in **giorno**, **mese** e **anno** in numero.
2. Generare e stampare a video una password concatenando **nome**, **cognome**, **colore preferito** e **somma** di **giorno**, **mese** e **anno di nascita**, separate dal carattere `-`.

## Esempio
La password di un utente che si chiama **Pinco Pallo**, nato il **12/05/1994**, il cui colore preferito è il **magenta** sarà `Pinco-Pallo-magenta-2011`.
